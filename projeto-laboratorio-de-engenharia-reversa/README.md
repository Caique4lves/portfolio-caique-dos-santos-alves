## 📝 Descrição do Projeto
Este projeto é uma ferramenta profissional de personalização de QR Codes que permite transformar códigos estáticos em elementos de design únicos. O objetivo principal é oferecer total controle criativo, permitindo ajustes em tempo real de cores, formas e logotipos para alinhar o código à identidade visual de qualquer marca ou projeto.

Desenvolvido para ser robusto e intuitivo, o sistema processa configurações complexas de estilização (como gradientes, formatos de pontos e detecção automática de versão), garantindo que o QR Code final seja não apenas visualmente atraente, mas perfeitamente legível por dispositivos móveis.

## 🚀 Tecnologias Utilizadas
* **Core:** React 19 & TypeScript
* **Estilização:** Tailwind CSS (v4)
* **Motor de Renderização:** `qr-code-styling`
* **Iconografia:** `Lucide React`
* **Build Tool:** Vite

## 📊 Resultados e Aprendizados
O projeto evoluiu para uma solução de alta performance, focada em experiência de usuário e estabilidade.
* **Customização Dinâmica:** Implementei um sistema de estados complexo que permite alternar entre modos de dados (Byte, Kanji, etc.) e versões de QR Code com feedback visual imediato.
* **Resiliência do Sistema:** Aprendi a aplicar **Error Boundaries** customizados para capturar falhas de configuração em tempo real, evitando que a aplicação trave caso uma combinação de dados e versão seja tecnicamente impossível.
* **Tratamento de Arquivos:** Implementei um pipeline de processamento de imagens para logotipos, garantindo que o upload de arquivos externos seja integrado de forma segura ao preview do QR Code.

## 🔧 Como Executar
1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Inicie o servidor de desenvolvimento: `npm run dev`.
